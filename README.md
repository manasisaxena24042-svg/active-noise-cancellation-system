# Active Noise Cancellation System

This project explores an analog Active Noise Cancellation (ANC) system using destructive interference.

## Overview
- Designed using analog circuit concepts
- Simulated using MATLAB and LTspice

## Key Concepts
- Effect of delay (τ) on phase mismatch
- Relation: φ = 2πfτ
- Reduction in noise cancellation due to delay

## Approach
- Implemented amplitude scaling to compensate for delay-induced errors
- Analyzed system performance using frequency-domain (FFT) techniques

## Tools Used
- MATLAB
- LTspice
